CSS

Les CSS pour “Cascading Style Sheets” sont le code utilisé dans le web pour mettre en forme une page, leurs donner couleur, forme et animation. le ‘3’ est un numéro de version correspondant a l’ensemble des fonctions disponibles.

Je n’ai pas eue besoin de me pencher avec attention sur ce langage qui est a la portée de tous. C’est a force d’utilisation que l’on apprends le nom des propriétées et ne perdons pas de temps a devoir les chercher.

Mais c’est certainement le langage qui renvoi le plus de visuel. Ayant un attrait pour les concequences de mon code, je trouve jouissif de sauvegarder une page css car je vois de suite les modifications apportées sur mon écran. Quel bonheur…
